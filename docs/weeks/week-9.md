## Exercise

Build a fact-checking agent using Chain-of-Verification:
1. Decompose claims into atomic facts
2. Generate verification questions
3. Answer questions independently
4. Check consistency with original claims

## Discussion Questions

1. What types of hallucinations are hardest to detect?
2. How should agents express uncertainty?
3. Can verification be done efficiently at scale?
