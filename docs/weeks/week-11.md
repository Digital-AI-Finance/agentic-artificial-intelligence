## Exercise

Build a domain-specific agent for one of:
1. **Code Generation**: Build an agent that writes and tests code
2. **Research Assistant**: Build an agent that searches and summarizes papers
3. **Financial Analysis**: Build an agent that analyzes market data
4. **Customer Support**: Build an agent that handles support tickets

## Discussion Questions

1. How do domain constraints affect agent design?
2. What safety measures are needed for high-stakes domains?
3. When should agents defer to humans?
