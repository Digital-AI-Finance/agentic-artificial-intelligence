## Discussion Guidelines

### Do

- Search existing discussions before posting
- Use clear, descriptive titles
- Include relevant code snippets or error messages
- Tag your posts with appropriate labels
- Be respectful and constructive
- Share your solutions when you solve a problem

### Don't

- Post solutions to assignments during active periods
- Share API keys or sensitive information
- Spam or post off-topic content
- Use hostile or dismissive language

## Getting Help

### For Technical Issues

**Best for**: Code errors, notebook bugs, environment problems

1. Check the [FAQ]({{ '/faq' | relative_url }}) first
2. Search existing [GitHub Issues](https://github.com/Digital-AI-Finance/agentic-artificial-intelligence/issues)
3. Post on [Q&A Discussions](https://github.com/Digital-AI-Finance/agentic-artificial-intelligence/discussions/categories/q-a)

**Include**:
- Your Python version and OS
- Error message (full traceback)
- Steps to reproduce
- What you've already tried

### For Conceptual Questions

**Best for**: Understanding concepts, clarifying theory

1. Check lecture slides and notebooks
2. Review the [Glossary]({{ '/glossary' | relative_url }})
3. Post on the relevant week's discussion

### For Course Logistics

**Best for**: Deadlines, grading, enrollment

- Email the instructor directly
- Check the [Syllabus]({{ '/syllabus' | relative_url }})

## Community Resources

### Research Collaboration

Looking for research collaborators? Post in the [Ideas](https://github.com/Digital-AI-Finance/agentic-artificial-intelligence/discussions/categories/ideas) category with:

- Your research interest
- Your background
- What you're looking for

### Project Showcase

Built something cool? Share it in [Show & Tell](https://github.com/Digital-AI-Finance/agentic-artificial-intelligence/discussions/categories/show-and-tell):

- Project description
- Key techniques used
- Demo or screenshots
- Link to code (if public)

### Study Groups

Interested in forming a study group? Post in [General](https://github.com/Digital-AI-Finance/agentic-artificial-intelligence/discussions/categories/general) with:

- Your timezone
- Topics you want to focus on
- Preferred meeting format
